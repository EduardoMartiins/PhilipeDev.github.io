Dúvidas frequentes

Perguntas que todo cliente costuma fazer antes de criar um site.

Preciso pagar hospedagem?

Para sites simples, consigo publicar em uma estrutura gratuita inicialmente. Caso você queira domínio próprio, o custo do domínio fica por conta do cliente.

O site funciona no celular?

Sim. Todos os sites são feitos para funcionar bem em celular, tablet e computador.

O site tem botão para WhatsApp?

Sim. O botão de WhatsApp é colocado em pontos estratégicos para facilitar o contato dos clientes.

Em quanto tempo o site fica pronto?

Sites de uma página normalmente ficam prontos em poucos dias, dependendo do envio das informações, fotos e ajustes solicitados.

Posso pedir alteração depois?

Sim. Pequenos ajustes podem ser combinados. Também existe a opção de manutenção mensal para atualizações frequentes.
